CSS selectors you should know:

  1. * - selects ALL
  2. #myId - select element with id='myId'
  3. myClassName - select all elements with class ='myClassName'
  4. li - select all elements with tag li
  5. .parent .child - nested selector, will select all elements with class='child' nested inside element with class='parent'
  6. .class.class - select all elements with class='class1 class 2'
  7. .my_el + .el - select element with class='el' placed after element with class='my_el'
  8. .parent > .child - select element with class='chils' which is a 1st lvl child of a .parent
